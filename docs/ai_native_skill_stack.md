# AI-Native Skill Stack Roadmap

This project adopts a 10-domain capability model for Codex-era systems:

1. AI-native software engineering
2. Agent and tool orchestration
3. Event-driven and streaming architectures
4. Distributed systems thinking
5. Autonomous testing and verification
6. Multimodal data engineering
7. Human-AI interface design
8. Security for AI-generated systems
9. Self-evolving codebases
10. Systems thinking and meta-architecture

## Mapping to `qso-fabric`

- AI-native engineering: schema-first, typed services, MCP-facing API
- Agent orchestration: MCP tools + runtime composition
- Streaming architecture: event log + async subscriptions + entanglement propagation
- Distributed thinking: GDML modules and node registry stubs
- Autonomous verification: executable tests and replay/rollback primitives
- Multimodal readiness: QFF block design for tensors/waves and extensible state schema
- Human-AI interface: URI-first object model and intent-level operations
- Security: signed events and snapshot signatures via crypto service
- Self-evolving code: modular services for iterative replacement/upgrades
- Meta-architecture: explicit layering and service boundaries
